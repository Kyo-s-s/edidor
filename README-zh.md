# 安装
在hugo网站根目录运行:

```
cd themes
git clone https://github.com/jacobsun/edidor
```
或者(推荐这种方式)

```
cd themes
git submodule add https://github.com/jacobsun/edidor
```

# 使用
复制 `exampleSite/config.toml` 到网站根目录, 运行`hogo server`预览.
